## 🧠 Project Overview & Intent
This project was built during my gap year to explore the capabilities of **Prompt Engineering** in software development. Rather than traditional manual coding, I acted as the Technical Lead and AI Architect—structuring the project, designing the system prompts, and systematically debugging AI-generated outputs to create a production-ready interface.

## 🛠️ Tech Stack & AI Tools
* **Core Technologies:** HTML5, CSS3 , JavaScript
* **AI Coding Engine:** Google Gemini Pro
## 🎯 Prompt Engineering Workflow
To achieve the design accuracy of the original Backdrop platform, I utilized a multi-step prompting strategy:
1. **Component Scaffolding:** Prompted the AI with a structural breakdown of the layout to ensure semantic structure.
2. **Iterative Styling:** Used descriptive UI/UX prompting to fine-tune alignment, spacing, and responsive behaviors.
3. **Contextual Debugging:** When the AI generated errors, I provided console logs and targeted feedback to guide the AI to the correct solution.

## 💡 Key Learnings
* Mastering iterative and contextual prompting for complex code generation.
* Understanding core web layout principles (Flexbox/Grid) to critique and fix AI code.
* Project management, asset handling, and deployment workflows.
